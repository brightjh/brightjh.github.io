# IDEA 常用快捷键

  1. 代码编辑相关：
    • 格式化代码：Ctrl + Alt + L
    • 补全代码：Ctrl + Space
    • 快速生成代码（如生成构造函数、getter/setter等）：Alt + Insert
    • 查找类、文件或符号：Ctrl + N
    • 查找方法：Ctrl + F12
    • 查找替换：Ctrl + R
  2. 代码导航和编辑器操作：
    • 跳转到定义处：Ctrl + 左键点击
    • 返回上一次编辑的位置：Ctrl + Alt + 左箭头
    • 快速切换到上一个/下一个编辑位置：Ctrl + Alt + 左/右箭头
    • 选中当前行：Ctrl + Shift + 上/下箭头
    • 复制当前行或选中的代码块：Ctrl + D
    • 删除当前行或选中的代码块：Ctrl + Y
  3. 重构操作：
    • 重命名：Shift + F6
    • 提取方法：Ctrl + Alt + M
    • 提取变量：Ctrl + Alt + V
    • 提取常量：Ctrl + Alt + C
  4. 运行和调试：
    • 运行应用程序：Shift + F10
    • 停止应用程序：Shift + F9
    • 调试应用程序：Shift + F9（启动调试会话）和 F9（继续执行）
    • 在断点和断点之间切换：Ctrl + Shift + 上/下箭头
